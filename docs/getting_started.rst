Getting Started
===============

This document will show you how to get up and running with Pymeerkat.

Installation
---------------
1. Install the external dependencies
    * ffmpeg
    * opencv
On Mac OSX use
.. code-block:: bash
    brew install ffmpeg opencv

On Ubuntu use
.. code-block:: bash
    sudo apt-get install ffmpeg opencv

2. Install library
.. code-block:: bash
    sudo python setup.py install
